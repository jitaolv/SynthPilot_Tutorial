- 在D:\youFPGA\cc_learn\led下创建以下目录：

  ```
  \prj
  
  \xdc
  
  \rtl
  ```

- 在\prj目录下创建一个vivado工程，芯片型号：XC7Z020-CLG4841，工程名为demo01
- 在\rtl下创建一个blink led模块并添加到工程
- 在\rtl下创建一个按键模块(带消抖处理)到工程
- 在\rtl下创建一个顶层模块，例化上面编写的两个模块，将此模块设为顶层
- 在\xdc下按照此文件末尾的引脚信息，创建约束，并添加到工程
- 关闭工程

```
灯0: T12
灯1: U12
灯2: V12
灯3: W13

按键: M20

50M时钟: K17

电平标准:3.3v
```
